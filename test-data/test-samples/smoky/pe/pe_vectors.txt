
Incorrect_DosSignature.bin
  Цель:
    Отвергнуть файлы без DOS-сигнатуры
  Описание:
    В этом файле выставлена не правильная DOS сигнатура. Эта сигнатура
    ищется по нулевому смещению от начала файла.
  Кратко:
    Выставлено  : 00000000 : 304D
    Должно быть : 00000000 : 5A4D


Incorrect_NtSignature.bin
  Цель:
    Отвергнуть файлы без сигнатуры NT-заголовков
  Описание:
    В этом файле выставлена не правильная сигнатура для NT заголовка.
    Смещение этого заголовка вычисляется по полю IMAGE_DOS_HEADER->e_lfanew
    от начала файла.
  Кратко:
    Выставлено  : 000000F8: 00003550
    Должно быть : 000000F8: 00004550


Incorrect_OptHdrMagicVal32.bin
  Цель:
    Отвергнуть файл без сигнатуры 32 или 64 битного дополнительного NT-заголовка.
  Описание:
    В число NT заголовков входит дополнительный(англ. Optional Header).
    Его расположение сраву же за основным(англ. FileHeader). Используя WinNt.h
    до него можно добраться как &peFileOffset -> IMAGE_NTHEADERS32->OptionalHeader
  Кратко:
    Выставлено  : 00000100: CCCC
    Должно быть : 00000100: 010B
