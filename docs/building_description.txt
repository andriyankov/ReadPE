
Документ по сборке и настройке различных проектов

#
# Общие сведения:
#

  * Требуется установить Python ветки 3.x. В данный момент использую 3.2-amd64
  * Используется MS Visual Studio версий 2010 и 2012, возможно будет работать и для 2008
  * В тексте используется метапеременная $(SolutionDir). Она обозначает файловый путь 
    к директории с проектами(libraries, readpe, etc)

#
# Общий список действий
#

 0. Скомпилировать и настроить Boost-библиотеки
 1. Скомпилировать python-пакет $(SolutionDir)\readpe
 2. Настроить проекты в $(SolutionDir)\readpe.sln

#
# 0. Boost.Libraries
#

 0. Скачать архив библиотеки с офиц. сайта по ссылке: http://www.boost.org/
 1. Скомпилировать b2.exe в корневой папке библиотек с помощью bootstrap.bat.
    (этот шаг, возможно, будет лишним, если в батнике из п.2 вызывается bootstrap.bat)
 2. Положить $(SolutionDir)\tools\boost_compile.cmd в корень папки boost-библиотек
 3. Запустить батник
 4. В глобальных настройках в MSVS указать файловые пути до папок:
 4.1. Корневая папка boost-библиотек(к примеру вот : d:\third-party-libraries\boost_1_53_0\)
 4.2. Скомпилированные *.lib файлы boost-библиотек для 32- и 64-битных версий (вот пример пути 
    к 32-битным: d:\third-party-libraries\boost_1_53_0\stage32\lib\ )

#
# 1. 'readpe' python-пакет
#

Предусловия: Установлен Python версии 3.2 и выше

 0. Перейти в папку $(SolutionDir)\py-package\
 1. Запустить команду python.exe .\setup.py install

#
# 2. libraries
#

Предусловия: 
  * Установлен набор библиотек "0. Boost.Libraries"
  * Установлен py-пакет "1. 'readpe' python-пакет"

#
# 3. readpe
#

Предусловия: Настроена сборка "1. libraries"
